##  Mac OSX:

```bash
brew install python3 pyqt
pip3 install pyqtgraph numpy scipy

brew install portaudio
pip3 install pyaudio

```

## Linux:

```bash
sudo apt-get install -y \
  python3 \
  python3-numpy \
  python3-pyqt4 \
  python3-pyqtgraph \
  python3-pyaudio \
  python3-scipy

```

# Running

```bash
./visualize-harmonics

```
